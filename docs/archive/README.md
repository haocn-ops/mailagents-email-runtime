# Archive

This directory stores historical records that are useful for audit trails,
rollout history, and dated verification notes, but should not stay mixed into
the main long-lived guides.

Use this directory for:

- dated environment verification logs
- rollout records tied to a specific maintenance window
- historical operational notes that may become stale
- superseded guidance kept only for reference

Do not use this directory for:

- the current local setup flow
- the current deployment checklist
- the current production runbook
- stable API, auth, or compatibility guidance
